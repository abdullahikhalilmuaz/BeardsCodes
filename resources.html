Introduction:

HTML is the standard markup language for creating web pages.

What is HTML ?

  • HTML stands fo Hyper Text Mark Language
  • HTML is the standard markup language for creating wep pages
  • HTML describes the structure of a web page
  • HTML consists of a series of elements
  • HTML elements tell the browser how to display the content
  • HTML elements lebel pieces of content such as "this is a heading " , " this is a paragrah" , "this is a link", e.t.c

A Simple HTML Document

Example

<!Doctype html>
<html> <head>
<title> Page Title </title>
</head>
<body>
<h1> My first Heading<h1>
<p> My first Paragraph<p>
</body>
</html>
copied successfuly!
Editor ?

Example Explained

  • The <!Doctype html> declaraton defines that this document is an HTML5 document.
  • The <html> element is the root of an HTML page.
  • The <head> element contains meta information about the HTML page (which is shown in browser's totle bar or in the page's tab).
  • The <body> element defines the document's body and is container for all the visible contents such as headings, paragraphs , images , hyperlinks , tables , lists e.t.c.
  • The <h1> element defines a large heading
  • The <p> element defines a paragraph

Headings (h1-h6)

HTML headings are defined with the <h1> to <h6> tags

<h1> defines the most important heading , <h6> defines the least important heading.

Images

Images can improve the design and the apprearance of a web page .

HTML Image syntax
The HTML <img> tags is used to embed an image in a web page.

Images are not technically inserted into a wb page ; images are linked to web pages. the <img> tag creates a holding space for the references image.

The <img> tag is empty , it contains attributes only , and does not have a closing tag.

The <img> tag has two required attributes :

  • src = Specifies the path to the image
  • alt (optional) = Specifies the alternate text for the image , incase the image did'nt load , the user will se alternate text instead , sound funny ha ??, absolutely let's see an example

Syntax :
<img src="url" alt="alternatetext">


Example :

Copy paste in the web Editor , located in the Nav Bar.

Example

<!Doctype html>
<html> <head>
<title> Page Title </title>
</head>
<body>
<img src="image.png" alt="Cofee Image">
</body>

Image Size - Width and Height

You can use the style attribute to specify the width and the height of an image , e.g :

<!Doctype html>
<html> <head>
<title> Page Title </title>
</head>
<body>
<img src="image.png" alt="Cofee Image" style="width: 300px; height: 150px;">
</body>
</html>
